Building the Handheld Diagnostics Tool

POSTED: MARCH 5, 2026 • 8 MIN READ

Developing a handheld tool for a robot requires a balance between low-level signal integrity and high-level user feedback. In this post, I'll break down the SPI communication protocol used to drive the status display.

Hardware-in-the-Loop Testing

One of the primary requirements for the Aegis Controller was the ability to perform manual overrides without relying on the primary Wi-Fi link. This meant building a direct physical bridge using an ESP32 and an STM32.
