Aegis Communication Protocol

A resilient, low-latency state synchronization protocol for dual-computer robotics platforms.

C++20 Linux Sockets Jetson Orin

Project Overview

The Aegis protocol was designed to solve the problem of failover latency in competitive robotics. When a primary flight computer (Jetson Orin Nano) experiences a hang or a ROS2 node crash, the backup (Jetson Nano) must take over the chassis control within 50ms to prevent collision.

Key Features

  • ▹
    Heartbeat Monitoring: 1000Hz health checks with sub-millisecond timeout detection.
  • ▹
    Differential State Updates: Only transmits changed bytes to minimize network overhead on the local switch.
